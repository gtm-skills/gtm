## How to Use

1. Pick your role
2. Find the workflow stage you're in
3. Copy the relevant prompts
4. Fill in the `[BRACKETED]` variables
5. Paste into Claude, ChatGPT, or any LLM

## What Each Playbook Includes

Every role playbook covers:

- **Daily Workflows** - Prompts for everyday tasks
- **Common Scenarios** - Templates for typical situations
- **Edge Cases** - How to handle tricky situations
- **Best Practices** - What good looks like
