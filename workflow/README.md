## How to Use

1. Pick the workflow that matches your situation
2. Follow the steps in order
3. Use the prompts at each stage
4. Fill in the `[BRACKETED]` variables
5. Move to the next step when ready

## Workflow Philosophy

Each workflow is designed to:

- **Be Sequential** - Steps build on each other
- **Be Complete** - Everything you need from start to finish
- **Be Practical** - Real prompts, not theory
- **Be Flexible** - Adapt to your specific situation
